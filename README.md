Проект JSTest (EpolSoft)

Проект из двух частей: backend и frontend, запускать отдельно командой npm start, перед запуском создать базу данных project(mysql).

Задание:
Реализовать GUI интерфейс (HTML5, CSS3, JavaScript, React/Redux) со следующим функционалом:
1.Возможность загрузить данные (кнопка Upload) из JSON файла (пример ниже) данные подгружаются в приложение и отображаются в виде таблицы.
2.Пользователь может отредактировать запись (кнопка Edit)
3.Пользователь может удалить запись (кнопка Delete), с предварительным подтверждением у пользователя (действительно ли он хочет удалить запись).
4.Пользователь может добавить запись (кнопка Add) - появлятеся новая пусткая строка в таблице с возможностью ввода данных.
5.Поля в таблице сортируются при нажатии на заголовок колонок таблицы.
6.Над таблицей есть поле для фильтрации списка пользователей (фильтр ищет совпадения по всем полям/колонкам таблицы).
7.При нажатии на кнопку Submit данные сохраняются в базу MongoDB или MySQL (на ваш выбор)
8.При нажатии на кнопку Export данные из таблицы выгружаются в файл в формате JSON.
P.S.: Использование Redux - обязательно.
